{}
{
Zaver的是一个基于事件驱动的高性能服务器，具备服务器的基本功能，编程模型是事件驱动 + 非阻塞I/O + 线程池。Zaver的特点是用非常少的代码展示了高性能服务器的核心结构，即可作为一个轻量级服务器使用，也可为开发者进一步学习网络编程打下基础。

    \item 项目地址: \url{https://github.com/zyearn/zaver}
\end{itemize}
